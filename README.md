# 👭 Friends Circle Database

> **"Friendship is one of life's greatest treasures, and every friendship has a story. This project represents those stories using SQL."**

A beginner-friendly **MySQL project** that demonstrates how to create and manage a database while storing information about my close friends. The project showcases fundamental SQL concepts through a real-life example, making learning both practical and enjoyable.

---

## 🌟 Project Highlights

✨ Real-life friends database

🎂 Birthday month tracking

🎓 Course information

❤️ Friendship score (out of 100)

🛠️ Hands-on SQL operations

📚 Beginner-friendly project

---

# 📖 Project Overview

This project was created to practice **SQL Database Management** using MySQL Workbench. Instead of using random sample data, I designed a database based on my own friends, making the project more meaningful and engaging.

The project demonstrates how SQL can be used to organize and manage real-world information efficiently.

- Understand database creation
